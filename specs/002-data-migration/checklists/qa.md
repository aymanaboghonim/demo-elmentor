# QA Checklist: Real Data Migration

- [ ] All migrated content matches the original site (text, images, links)
- [ ] Section order and structure are correct
- [ ] Bilingual toggle works for all new content
- [ ] No missing or duplicated data in any section
- [ ] All links (internal/external) work as expected
- [ ] All images display correctly in both languages
- [ ] Responsive layout verified on desktop, tablet, and mobile
- [ ] No layout breakage in RTL mode
- [ ] No console errors or warnings in browser
- [ ] Accessibility: alt text, color contrast, keyboard navigation
- [ ] All acceptance criteria from spec are met
