# Migration Checklist: Real Data Migration

- [ ] Source site is archived and accessible for reference
- [ ] All text content is extracted and saved to JSON files
- [ ] All images/assets are downloaded and optimized
- [ ] Data files are organized by section in `src/assets/data/`
- [ ] i18n files updated for all new content
- [ ] Section modules updated to use new data
- [ ] Asset/image paths updated in code and data
- [ ] All migrated data is reviewed for accuracy
- [ ] Migration scripts (if any) are documented
- [ ] Manual edits are tracked and reproducible
